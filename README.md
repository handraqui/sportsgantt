# sportsgantt
 A visual timeline of the world's sports calendar

Initial example of entry for sport discipline:

competition: Formula 1
season_start: 2027-03-14
season_end: 2027-12-05
sport: motorsport

## Goal!!!

Create a customizable yearly sports calendar showing:

- seasons
- playoffs
- championships
- major events
- Olympic sports

## Initial scope for next version

- Football (Top 10 leagues, CONMEBOL, FIFA and UEFA)
- Motorsport (F1)
- Basketball
- Baseball
- Tennis
- Volleyball
- Surf
- Skate
- Olympic sports
